A vulnerability was found in Google Android 10.0/11.0. It has been rated as problematic. This issue affects some unknown processing of the file tapjacking/overlay. The manipulation of the argument wifi.RequestToggleWifiActivity leads to improper restriction of rendered ui layers. The identification of this vulnerability is CVE-2022-20212 . It is possible to launch the attack on the local host. There is no exploit available. It is recommended to apply a patch to fix this issue.
